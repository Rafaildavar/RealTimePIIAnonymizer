# RealTimePIIAnonymizer

Необходима версия Python 12 или 11.

```bash
pip install -r requirements.txt
```

Файл `metrics.py` содержит функции для вычисления метрик Precision, Recall и F1-score.

Файл `masker.py` содержит функцию `mask_pii`, которая принимает текст и возвращает анонимизированный текст, заменяя личные данные на специальные токены.
