# RealTimePIIAnonymizer

Real-time masking of personal data in LLM requests for banking assistants.

## О проекте

В Банке используются различные ИИ-ассистенты и агенты, которые помогают клиентам быстро и эффективно решать задачи: от консультаций по карточным продуктам до построения персональной инвестиционной стратегии или планирования путешествий.

При обработке клиентских запросов могут использоваться персональные данные, такие как ФИО, номер телефона, email, паспортные данные, номер карты и адрес проживания. Передача этих данных в LLM создает риски безопасности и юридические ограничения.

**RealTimePIIAnonymizer** — это прототип системы, которая в режиме реального времени:
- обнаруживает персональные данные в пользовательском запросе;
- маскирует их перед отправкой в LLM;
- сохраняет полезный контекст запроса;
- поддерживает безопасное восстановление данных в ответе.

## Основные возможности

- real-time обработка запросов;
- mask/unmask pipeline;
- rule-based и heuristic detection;
- поддержка NAME, PHONE, EMAIL, PASSPORT, CARD, ADDRESS;
- безопасное восстановление ответа;
- оценка качества на размеченном датасете.
